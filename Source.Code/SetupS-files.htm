What is SetupS Suite?

SetupS Suite is a set of utility software to allow the silent installation of applications. The software can also create and edit silent install applications called 'apz' that can be installed on Windows operating system either individually or as a selection of files which is ideal for users to install multiple applications in one go.

  • SetupS Installer (as a ssApp-EXE package). (Recommended). Best for all occasions: new/fresh installs of SetupS or simply updating to the latest. Note: This is the same package the online self-updater uses.
    SetupS.SendTo.Suite_v#ProjectVersion#_ssApp.exe
    (mirror1) (mirror2) (mirror3 right click on file to download)
  • SetupS Installer (as an ssApp-APZ package). Remember to ONLY use ssWPI or ssGooey to install this. Especially good for either new/fresh installs of SetupS or simply updating to the latest. But DO NOT use the already-installed SetupS to update to this! It will fail. Instead, use the ssApp-EXE package (above).
    SetupS.SendTo.Suite_v#ProjectVersion#_ssApp.apz (mirror1) (mirror2) (mirror3 right click on file to download)
  • ssWPI update package (.7z). A slightly modified ppApp-type package for integration with the ssWPI or ssGooey project(s). Simply copy it to the ssWPI folder, extract "here", then delete it.
    SetupS.Uninstaller_v#ProjectVersion#.exe (mirror1) (mirror2) (mirror3 right click on file to download)
  • SetupS Installer (as an InnoSetup package). The vanilla installer for the SetupS Sendto Suite.
    SetupS.Project_v#ProjectVersion#_DevelopersPack.7z (mirror1) (mirror2) (mirror3 right click on file to download)
  • SetupS Uninstaller. Run this for a thorough and complete removal of SetupS. This will clean up such things as SetupS shortcuts and registry entries. Also, the removal of the Advanced Startmenus; but infrastructure-only. Which means you should use the SetupS.Control.Panel to change the Startmenu style back to Standard/default before uninstalling SetupS.
