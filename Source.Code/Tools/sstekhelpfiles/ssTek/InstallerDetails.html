Installer Details        


This is the necessary information for making an ordered list of installers (.exe .msi) or other actions that must occur in order to setup and create shortcuts for this application. The box at the bottom of this section is that list of actions and/or installers plus any silent switches accompanying them -- all in the order of occurrence.

  • Installer: For ssApps only. Use the browse button to select the installer or setup program. ssEditor will attempt to find the correct silent switch for that installer with its built-in Auto Silent Switch Finder (ASSF) but sometimes a best match is not always made1.

    Click the 'Add' button to add this information to the Installer List.

Example:

Installer: Setup.exe

  • Usage: For ssApps only. These are the silent switches to accompanying the installer above to force the setup program to do what is called an "unattended" setup. Click the 'Add' button to add this information to the Assembly List.

Example:

Usage: /S

____________________

1 Note: ASSF performs its function with these utilities in the following order of success:

    1. SKF - Silent Key Finder
    2. TrID - File Identifier
    3. USSF - Universal Silent Switch Finder
    4. PEiD - Portable Executable iDentifier
