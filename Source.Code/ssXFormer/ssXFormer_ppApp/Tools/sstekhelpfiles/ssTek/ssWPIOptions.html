ssWPI Options        


Some options pertaining to only to ssWPI:


Status: Optional.


Always show: If selected it will always show in the ssWPI list.

Always hide: If selected it will always hide from the ssWPI list.

At Setup only: If selected it will show only during setup.

Install T13: If selected will be installed at setup time of 13 Minutes remaining.

Run as x64: Redirect folder locations and reg entries, enabling this will treat it as a x64 OS and allow SetupS to access x64 locations. Note, this flag is both non-functioning and obsolete.

Install Instant: .

Priority Order: to install items from ssWPI selection screen.
