How to update a ppApp or ssApp        


Updating a ppApp or ssApp is even easier:

    1. If .apz then first extract to a temporary build folder.
    2. Add any updated files (for the ppApp) or installers (for the ssApp -- don't forget to delete the old one)
    3. Right-click, "edit" (to use ssEditor) or "open with" (something like Notepad2)
    4. Make changes/updates as needed
    5. Save or Build (if .apz this will also update the .md5)
