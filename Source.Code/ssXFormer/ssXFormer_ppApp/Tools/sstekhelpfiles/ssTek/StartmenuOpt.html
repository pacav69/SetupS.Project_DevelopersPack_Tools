StartMenu Styles        


Install an Advanced StartMenu.
Sets an Advanced StartMenu style by selecting from the drop box. Also, revert back to Standard (default) StartMenu if desired.

Sort (organize) Startmenu shortcuts.
Select this option to sort all existing shortcuts to the selected StartMenu style. Otherwise, only the StartMenu style is installed.


Keep empty Advanced Startmenu folders.
Select this option to install (and retain) the full selected Advanced Startmenu including empty folders.


Ξ Note: Click the "Okay" button to make the changes to the StartMenu.
