Fadertainer Options        


Use to set Fadertainer options globally for all SetupS tools.

    • Activate: Use to turn on (or off) the Fadertainer.
    • Always On Top: The Fadertainer will always show in the foreground.
    • During Installs Only: (ssRegenerator-only option)
    • Speed: Number of seconds to fade in or fade out.
    • Screen Location: Where the Fadertainer is to appear.
      (T=Top, L=Left, R=Right, C=Center, M=Middle, B=Bottom)
    • Fader Module: The addon (executable) to run in the background with Fadertainer.
      • RunRUNAway -- Fadertainer moves away from the mouse. Automatically exit after 8 second timeout.
