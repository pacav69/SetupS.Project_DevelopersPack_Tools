Shortcuts Options        


Skip shortcuts.
During processing, do not create or process any shortcuts for:

    • ssApps.
    • ppApps.
    • ppGames.

Remove pre-existing.
If shortcuts from the following already exist, then do not remove:

    • Root (Startmenu).
    • Programs (Startmenu).
    • Startup (Startmenu).
    • Pin (Startmenu).
    • Desktop.
    • Quick Launch.
    • SendTo.
    • Taskbar Pin.
