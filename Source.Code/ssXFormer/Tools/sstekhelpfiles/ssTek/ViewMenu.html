View Menu        


Current working folder...        (F2)

Open explorer view of the current working folder.

Current destination folder...

Open explorer view of the current destination folder.

Patch Folder

This folder gets copied into the Install-to folder once the install is completed.

Fader (.png)

A PNG image file that will to show on the fader screen, if enabled.

Icon (.ico)

The ICO image file will be used for startmenu and install-to folder icon.

ScreenShot (.jpg)

The JPG image file screen shot you will display in the ssWPI when the app/game title is selected.

Batch/Script file (.bat/.cmd)

A CMD (or BAT) file that contains an external script file.

Registry file (.reg)

A REG file that contains external registry entries to apply.

View %Variables% List        (F3)

Displays a pop-up of the Automatic Variable Substitutions Chart.

Source in Notepad                (F4)

View the actual .app or .ppg source file in Notepad.

Meta-data Editor

Opens the current file's meta-data to view or edit.

File Version info

Shows the current file's version information.
