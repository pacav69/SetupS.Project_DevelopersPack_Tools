File Menu        


New                (CTRL + N)

Create a new .app or .ppg file.

Open                (CTRL + O)

Open an existing .app or .ppg file for editing.

Save                (CTRL + S)

Save the current .app or .ppg file.

Ξ Note: This will also convert strings to variables and convert between BuildTypes. For example, "C:\Program Files" becomes %ProgramFiles% and changes an ssApp to a ppApp (or vise-versa) including all the auxiliary support files and folders.

Save As                

Save the current .app or .ppg file to a different folder.

Ξ Note: This will also convert strings to variables and convert between BuildTypes. For example, "C:\Program Files" becomes %ProgramFiles% and changes an ssApp to a ppApp (or vise-versa) including all the auxiliary support files and folders.

Exit

Leave the SetupS Editor application.
