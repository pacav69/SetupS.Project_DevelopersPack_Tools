ssControlPanel Commands        

ssControlPanel can be invoked via the command line as follows:


Usage: ssControlPanel [<switches>...]

Alias: SetupSCP


<Switches>

Except where noted, these settings affect the defaults for all SetupS tools.


Startmenu:

  -LastOS|Kazz: Force or change to a different SetupS advanced Startmenu sorting-structure(s).

  -Standard: Force or change to a standard/non-SetupS Startmenu sorting-structure(s).

  -MenuInstallOnly: Do not sort (organize) any Apps/ppGames shortcuts; just the Startmenu structure gets installed.

  -ReSort: Force sorting (organize) of Apps/ppGames using the current Startmenu sorting preference.

  -ssClean [=Off][,StartMenuOnly][,AllShortcuts][,DudInstallPaths][,NotDudInstallPaths][,DeepScan][,NoDeepScan]:

       Perform shortcut cleaning and whether to limit the cleaning to never, or only the StartMenu dud shortcuts or to  include all shortcut locations such as Desktop, Sendto, & Quicklaunch. Also whether or not to remove the install-to paths referenced by dud shortcuts or to do a deep-scan of uninstalled apps or games. Multiple settings are comma-separated, like so:

         -ssClean=StartMenuOnly,NoDeepScan


ppDrives:

  -ppAppsDrive=[x]: Sets desired ppApps Drive to x. Over-rides any current setting.

  -ppGamesDrive=[y]: Sets desired ppGames Drive to y. Over-rides any current setting.

  -ppDrivesScan[=Backwards|Forwards]: Scan and set for ppDrives. Scan direction can also be specified.

  -ppDrivesRegen: Runs ssRegenerator on ppDrives.

  -AllDrives: A ssRegenerator-only option for scanning all drives. Caution: this enables scans of portable and mapped network drives!


Reporting / notifications:

  -SplashOnTop=Yes|No: "Always On Top" option for Splash Text.

  -Verbose: Show progress as both splash text and tray-balloon tips.

  -Mute|Silent|Quiet: Totally silent install; hides everything including system tray icon and balloon tips.

  -TrayOnly: Show progress only as balloon tips from the tray. Must have balloon tips enabled!

  -SplashOnly: Show progress only as splash text/message boxes and hides both the tray icon and any balloon tips from the tray.


Process without shortcuts:

  -ssAppsShortcutsSkip: Do not create or process any shortcuts for ssApps.

  -ppAppsShortcutsSkip: Do not create or process any shortcuts for ppApps.

  -ppGamesShortcutsSkip: Do not create or process any shortcuts for ppGames.


Logging ("global" settings for all SetupS tools):

  -Log: Creates activity logs. Files created are in the form "ReportingConsole.000x.txt".

  -Debug: Turns on Debug-mode. Files created are in the form "DebugConsole.000x.txt".


Fadertainer ("global" settings for all SetupS tools):

  -Fadertainer=On|Off : Activate or deactivate the Fadertainer.

  -FaderSpeed={seconds} : Number of seconds to fade in or fade out. Default=.5. IE, -FaderSpeed=.3

  -FaderLocation=TL|TC|TR|ML|MC|MR|BL|BC|BR : T=Top, L=Left, R=Right, C=Center, M=Middle, B=Bottom

  -FaderOnTop=Yes|No : Fadertainer Option for use "Always On Top".

  -FaderModule={executable} : Module to run in the background with Fadertainer.

  -FaderInstallsOnly=Yes|No : Fadertainer Option for use "During Installs Only" (ssRegenerator-only option).


Options ("global" settings for all SetupS tools):

  -SafeInstall=On|Off : (depreciated/disabled)

  -SkipOSArch=Yes|No : SetupS/ssRegenerator option to process or skip apps/games based on the <Architecture> and/or <OS> information in an .app/.ppg file.

  -ssCleanOptions=Off[,StartMenuOnly][,AllShortcuts][,DudInstallPaths][,NotDudInstallPaths][,DeepScan][,NoDeepScan]:

       Determines whether to limit the cleaning to never, or only the StartMenu dud shortcuts or to include all shortcut locations such as Desktop, Sendto, & Quicklaunch. Also whether or not to remove the install-to paths referenced by dud shortcuts or to do a deep-scan of uninstalled apps or games.

  -Path=Add|Del: Permanently adds (or removes) the SetupS path to/from the environment path variable.

  -ssUninstall=On|Off|Fix|Refresh|Force : Activate/Deactivate the extra post-uninstall cleaning feature. Use "Refresh" or "Force" to update all to the current setting, "Fix" to revert everything without the feature (Note, "Fix" is the equivalent of "-ssUninstall=Off,Force").

  -Wait|NoWait: Depends whether or not you need to hold onto the SetupS(x86) process and return to it after SetupS(x64) is finished. Dual-architecture ppApp-mode only. (Default is 'Wait').

  -KillLauncher={NameOfLauncher}|KillProcess={NameOfProcess}: If you need to close a launcher or process. Especially useful if using the 'NoWait' switch above.


Auto Update:

  -AutoUpdate=On|Off : Automatic online updates of the SetupS Sendto Suite using one of the Repo links, below.

  -Repo={Link} : Primary link to find "Update.ini"

  -BackupRepo={Link} :  Backup link to find "Update.ini" in case the primary one isn't working.


  -Help|About|?: This help file.


Usage example:

ssControlPanel -LastOS -Mute -ppGamesDrive=D:
