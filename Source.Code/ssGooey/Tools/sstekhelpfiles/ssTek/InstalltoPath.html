Install-to Path        


Destination path -- the actual physical (final) location of the app or game. Typically %ProgramFiles% for ssApps, %ppApps% for ppApps, and %ppGames\% for ppGames.


Status: Required.


Example of Usage:

%ProgramsFiles%\MyApp

%ppGames%\MyGame

Ξ Note: Strings such as 'C:\Program Files\' will automatically be converted to %ProgramFile% for ssApps, %ppApps% for ppApps, and %ppGames% for ppGames. And vise versa.
