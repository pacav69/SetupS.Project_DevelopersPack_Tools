Recommended 7zip archive settings        

After many long discussions and a lot of trial & error, the team recommends the following settings to use with 7zip, especially if one has the computing power:

         "Install" archive   Archive deployment (.apz/.pgz)
ssApp:   N/A                 LZMA2,Ultra,Non-solid
ppApp:   LZMA2,Ultra,Solid   Copy/Store*
ppGame:  LZMA2,Ultra,Solid   Copy/Store*


Ξ Note that non-solid is not an option for Copy/Store since there is no compression algorithm involved.
