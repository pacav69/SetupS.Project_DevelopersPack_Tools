The Portable Executable (PE) format is a file format for executables, object code, DLLs, FON Font files, and others used in 32-bit and 64-bit versions of Windows operating systems. The PE format is a data structure that encapsulates the information necessary for the Windows OS loader to manage the wrapped executable code.

PEiD is an intuitive application that relies PE signatures to detect compilers, packers, and cryptors found in PE files – its detection rate is higher than that of other similar tools since the app packs more than 600 different signatures in PE files - while also featuring a HEX viewer and a task manager.

PEiD comes with three different scanning methods, each suitable for a distinct purpose. Regardless of the chosen type, the generated results are as accurate as possible due to PEiD’s error control method.
