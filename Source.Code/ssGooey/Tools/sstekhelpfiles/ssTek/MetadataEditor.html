Meta-data Editor        


Use drop-down box to select the desired Tags then click "Add" to add it to the Tags list.

  • Additional App/Game details:
    • Publisher: Original author or developer of the app/game.
    • Language: Language (if not English).
    • Rating: Use ratings between 1 to 5.
    • Additional: Extra information or comments.
    • Release Version: The version of publication by the original developer(s).
    • Release Date: The date of publication by the original developer(s). Recommended format: yyyy-mm-dd.
    • Required Runtimes: Prerequisites that need installed (such as, DirectX, Java, Flash, etc.).
    • Builder: The person or group who prepared this SetupS package.
    • Installed Size: The approximate size on the drive this package will ultimately require.
    • License Type:
      • Unknown: Unassigned.
      • Commercial: Non-gratis (Non-Free) proprietary/closed-source.
      • Non-profit: Gratis-only (Free as in beer) proprietary/closed-source.
      • Open source: Libre (Free as in speech and beer) non-proprietary.
