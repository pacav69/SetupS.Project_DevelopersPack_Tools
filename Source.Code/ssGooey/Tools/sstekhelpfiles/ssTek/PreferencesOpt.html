Preferences        


Default reporting / notifications.

    • Show Everything.
      "Verbose" setting. Show progress as both splash text and tray-balloon tips.
    • Hide Everything:
      "Mute" setting. Totally silent install; hides everything including system tray icon and balloon tips.
    • Tray Only.
      Show progress only as balloon tips from the tray. Must have balloon tips enabled!
    • Splash Only.
      Show progress only as splash text/message boxes and hides both the tray icon and any balloon tips from the tray.
    • Always On Top (Splash)
      The splash text remains visible no matter which window has focus.

Logging.

    • Reporting.
      Creates activity logs in the SetupS folder. Files created are in the form "ReportingConsole.000x.txt".
    • Debugging.
      Turns on Debug-mode. Creates debug logs in the SetupS folder in the form "DebugConsole.000x.txt".

Options.

    • SafeInstall.
      (depreciated/disabled)
    • SkipOSArch.
      SetupS/Regenerator option to process or skip apps/games based on the <Architecture> and/or <OS> information in an .app/.ppg file.
    • ssUninstall.
      Option for automatic ssCleaning after uninstalling an App or Game that was installed with SetupS.

DualArch Installs. Set the default for which type of DualArch that ppApps are supposed to assume during install-mode. See DualArch-Installers.

    • Force Type 2a.
    • Force Type 2b.

Updates.

    • Auto Update.
      Automatic online updates of the SetupS Sendto Suite using one of the Repo links, below.
    • Primary.
      Primary link to find the "Update.ini" file online.
    • Backup.
      Backup link to find "Update.ini" in case the primary one isn't working.
