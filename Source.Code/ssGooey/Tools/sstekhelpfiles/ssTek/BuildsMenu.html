Builds Menu        


Exit After Build

Closes SetupS Editor after finishing building a deployment package.

Calculate MD5 checksum

Creates the .md5 checksum for deployment packages.

Delete Source (folder/files)

After successfully building the deployment package, this option deletes any archived files.

Make Archive Deployment package (.apz/.pgz)

Creates the online distribution package rather than an ssWPI one (Folder Deployment package).

Webserver-friendly Filenames (.apz/.pgz)

Uses filenames for online distribution packages that won't conflict with webservers' filenaming conventions; such as having ".php" located anywhere in the filename not just at the end.

Use RAR

Create archives and files using RAR instead of 7z.

Ξ Note: Because of licensing, RAR.exe is not, nor will be, included in the Tools folder -- one must add it on their own recognizance. Fortunately, this is easy to do. Simply copy one file:

"C:\Program Files\WinRAR\Rar.exe" to "C:\Program Files\SetupS.SendTo\Tools".
