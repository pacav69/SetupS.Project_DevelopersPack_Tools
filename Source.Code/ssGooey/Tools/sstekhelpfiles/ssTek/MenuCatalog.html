Menu Catalog        


A recent new feature of SetupS is its ability to locate or sort shortcuts based on a catalog of categories rather than explicitly specifying its location. This allows for any number of custom designed Startmenu structures besides the two already included: LastXP and Kazz.

Simply select one or more categories from the Menu Catalog dropbox and click "+" add button to include it in the list.
